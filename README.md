# My Coffee Shop ☕

A simple and aesthetic coffee shop website created using only HTML.
This project showcases different coffee varieties, café-style sections, menu tables, coffee combos, contact pages, and smooth internal navigation.

## Features

* Coffee Menu
* About Section
* Café Hours
* Contact Page
* Coffee Combos
* Clean & Simple UI
* Semantic HTML Structure

## Built With

* HTML5

## References

* Inspired by Starbucks website design
* Learned concepts from Qspiders my training institute & YouTube tutorials
* Coffee history information taken from Google sources

## Purpose

This project was created for practicing frontend web development and improving HTML structure, layout design, and webpage organization skills.
